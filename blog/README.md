## 📖 專案簡介 (About)

> **Ⲙ𝔬ⲃ¡ⳝ𝔭ⲁ𝔠ⲉ (莫比空間)** 透過 PWA 技術與響應式設計，將部落格、實用工具庫、音樂頻道及多媒體服務無縫整合，打造一站式的數位導航入口。

## 🏗️ 技術架構 (Architecture)

本專案採用 **Vue.js** 單頁面應用架構，並針對 GitHub Pages 進行靜態優化：
* **目錄分層**：
    * `/` : RWD 品牌入口
    * `/blog` : SEO 友善靜態內容區
    * `/tol` : JSON 驅動之工具引擎
    * `/p` : 多媒體子系統核心
    * `/data` : 系統公告與全局設定檔
