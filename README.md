## 🏗️ 技術架構與開發 (Architecture)

本專案基於 **Vue.js** 單頁面應用 (SPA) 架構構建，結合 GitHub Pages 的靜態部署優勢，實現高效的加載速度與 SEO 友善度。



### 目錄結構說明
* ` / ` (Root) - 採用 RWD 響應式佈局的品牌入口。
* ` /blog ` - 專為搜尋引擎優化的靜態內容區。
* ` /tol ` - 單頁整合式工具箱。
* ` /p ` - 採用 JSON 驅動的多媒體與子服務系統。
* ` /data ` - 儲存公告（`notice.md`）與系統設定檔。
* ` /assets ` - 品牌識別 (BI) 與視覺資源管理。

## 🛠️ 維護指南

若需更新網站公告或調整工具內容，請直接編輯以下檔案：
1. **公告系統**：`/data/notice.md`（支援遠端同步與隨機迎賓詞）。
2. **工具列表**：`/tol/data.json`。
3. **子頁面配置**：`/p/sub.json`。
