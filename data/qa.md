### 🚀 [入門] 莫比空間是什麼樣的平台？
莫比空間（Ⲙ𝔬ⲃ¡ⳝ𝔭ⲁ𝔠ⲉ）是一個致力於數位極簡主義與高效能體驗的整合門戶。我們將音樂播放器、短影音（Shorts）、開發者工具庫與個人網誌無縫整合。透過 PWA 技術，我們提供如同原生 App 般的流暢度，且所有核心功能皆保持透明與安全性。

### 📱 [入門] 如何將莫比空間安裝為手機 App？
為了獲得沉浸式體驗，建議進行以下操作：<br>• **iOS (Safari)**：點擊下方「分享」圖示並選擇「加入主畫面」。<br>• **Android (Chrome)**：點擊右上角「...」並選擇「安裝應用程式」。<br>安裝後，系統將會隱藏瀏覽器網址列，並提供獨立的啟動圖示。

### ⚙️ [系統] 如何檢查版本與獲取最新功能？
您可以在頁面頂部的狀態列（如 v2.0 | SYNCING）確認連線狀態。莫比空間具備自動更新機制，當我們推送新功能時，Service Worker 會在背景下載。若遇到顯示問題，請嘗試「下拉刷新」或清除瀏覽器快取。

### 🛠️ [系統] 遇到程式錯誤（Bug）該如何處理？
請切換至 Wiki 頁面的「錯誤回報」分頁，點擊按鈕跳轉至 GitHub Issues 頁面。提交時請包含您的：1. 裝置型號（如 iPhone 15） 2. 瀏覽器 3. 錯誤截圖。這將大幅加快我們的修復進度。

### 🎵 [影音] 為什麼媒體播放會突然中斷？
這通常受作業系統的電源管理影響：<br>1. **省電模式**：iOS 或 Android 的省電模式會切斷背景網路連線。<br>2. **背景權限**：請確保瀏覽器具有背景播放權限。<br>3. **緩存中斷**：若網路不穩定，請嘗試重新整理頁面。

### 🧰 [工具] 工具庫 (/tol/) 是否會紀錄我的輸入？
絕對不會。莫比工具庫的所有處理過程（如 JSON 格式化、Base64 編碼）均在您的「本地瀏覽器」端執行。我們不傳輸、不紀錄、不儲存您的任何輸入數據，您可以放心處理敏感資訊。

### 👤 [用戶] 跨裝置資料同步的運作方式？
莫比空間使用 Supabase 雲端加密資料庫。只要您登入帳戶，所有的主題偏好、播放清單與個人化設置都會即時同步。無論是手機還是桌機，您的數位空間都能保持一致。

### 🔑 [權限] 為什麼需要存取 LocalStorage？
我們使用 LocalStorage 存取您的登入 Token 與主題設置。這能確保您下次開啟 App 時無需重複登入，並維持您上次使用的佈局配置。這是實現「秒開」體驗的核心技術。

### 🔒 [隱私] 莫比空間如何處理我的個人隱私？
我們遵循「隱私先行」原則：<br>• 我們不收集精確地理位置。<br>• 我們不讀取您的通訊錄或簡訊。<br>• 所有用戶數據皆受到 RLS（資料列級安全性）政策保護，僅限您本人存取。

### 🛡️ [安全] 關於資料庫連線金鑰的安全層級？
我們已完成 API 中繼站升級。所有的資料庫讀寫均透過 Vercel 雲端後端（Edge Functions）處理。網頁原始碼中不包含任何具管理權限的金鑰，有效防止任何惡意攻擊或資料篡改。

### ⚖️ [法律] 內容版權與社群行為規範
本站主要使用 Circum Icons 開源套件。用戶上傳或引用之影音、圖文內容，其版權歸原作者所有。本站嚴禁發佈違法或侵權內容，若經舉報核實，管理員有權在不預告的情況下刪除相關數據。

### 🔧 [開發] 如何參與開源計畫或貢獻代碼？
莫比空間是一個開源專案。您可以前往我們的 GitHub 儲存庫進行 Fork 並提交 Pull Request。我們長期歡迎 UI 優化建議、新工具開發以及多國語言翻譯的貢獻。
